Documentation for non-commands

The nosh toolset documents various things that are not commands per se.

Environment variables

  • TERM — the TERM, COLORTERM, and some other environment variables that determine terminal type for applications

Library functions

  • The TerminalCapabilities library determines terminal capabilities from the TERM and other environment variables.

File formats

  • cin — the CIN files that control how console-input-method operates

  • console-keyboard-map — the keyboard map files that are used by Human Input Device realizers for user-space virtual termunals

  • user-vt — the filesystem components of user-space virtual terminals

  • user-vt-realizer-configuration — user-space virtual terminal realizer configuration

Devices

  • freebsd-console — the FreeBSD /dev/console device (not to be confused with kernel virtual terminals, i.e. syscons(4) and vt(4))

  • linux-console — the Linux /dev/console device (not to be confused with kernel virtual terminals)

  • linux-vt — Linux kernel virtual terminals (not to be confused with the console device)

System administration

  • machine-id — the machine ID and the host ID derived from it
